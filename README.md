# 机2201-2班班级纪念网站

这是一个为机2201-2班制作的班级纪念网站，用来展示班级风采、宿舍生活、成长经历与毕业回忆。  
网站以照片、文字和音乐为主要内容，希望把大学几年的重要瞬间整理成一个可以长期保存和回看的线上纪念册。

## 网站地址

[https://hhhjssj.github.io/class-book/](https://hhhjssj.github.io/class-book/)

## 项目内容

网站主要包括以下几个部分：

- 班级首页展示
- 班级简介与风采
- 宿舍与集体荣誉展示
- 时光长廊与照片记录
- 毕业去向与未来展望
- 音乐与纪念氛围设计
